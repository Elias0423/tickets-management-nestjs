
## Description

  API built in NestJS for ticket management 

  
## Requirements
The following environment variables must be defined for the application to work correctly

```bash

PORT		--> Port on which the app will run, by default 3000
DB_HOST		--> Url where the database is hosted
DB_USERNAME	--> Database userÂ´s name
DB_USERPWD	--> Database userÂ´s password
DB_NAME		--> Database name

```

## Installation

  

```bash

$ npm install

```

  

## Running the app

  

```bash

# development

$ npm run start

  

# watch mode

$ npm run start:dev

  

# production mode

$ npm run start:prod

```

  

## Test

  

```bash

# unit tests

$ npm run test

  

# e2e tests

$ npm run test:e2e

  

# test coverage

$ npm run test:cov

```
